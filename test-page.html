Instructions for Testing

Test Scenarios:

1. Table Detection

  • Click "Detect Tables" - should find 3 tables
  • Click "Try Another Table" to cycle through them
  • Verify the preview shows correct Markdown table syntax

2. Custom Range Selection

  • Click "Select Custom Range"
  • Hover over different elements (they should highlight)
  • Click on "Test Content: Various Markdown Elements" section
  • Verify the Markdown includes headings, lists, code, etc.

3. Copy to Clipboard

  • Select any table or content
  • Click "Copy as Markdown"
  • Paste into a text editor or Heptabase
  • Verify the formatting is correct

4. Edge Cases

  • Try selecting small elements (like a single paragraph)
  • Try selecting large sections (like the entire page)
  • Try tables with special characters
  • Try nested lists and complex formatting
